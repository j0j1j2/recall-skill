# recall-skill

Claude Code 슬래시 커맨드 — 과거 작업 기록을 자연어로 조회합니다.

## Install

```bash
npx recall-skill
```

## Uninstall

```bash
npx recall-skill remove
```

## Usage

Claude Code에서:

```
/recall 어제
/recall 지난주
/recall 3월 20일
/recall 최근 3일
```

`~/.claude/history.jsonl`을 읽어 해당 기간의 작업 내용을 요약해줍니다.
